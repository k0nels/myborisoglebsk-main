Данный программный продукт представляет собой динамическое веб-приложение 
для управления городским контентом и взаимодействия с жителями.

1. ТЕХНОЛОГИЧЕСКИЙ СТЕК
-----------------------
- Язык программирования: PHP 8.1
- База данных: MySQL (СУБД MariaDB 10.x)
- Интерфейс: HTML5, CSS3 (Bootstrap 5), JavaScript (AJAX, Swiper JS, AOS)
- Среда разработки: OpenServer (OSPanel)

3. ИНСТРУКЦИЯ ПО РАЗВЕРТЫВАНИЮ (НА ХОСТИНГ)
-------------------------------------------
Для переноса сайта с локального сервера на реальный хостинг:

Шаг 1. Перенос файлов:
- Загрузите все файлы из папки проекта в корневую директорию хостинга (обычно public_html).

Шаг 2. Настройка Базы Данных:
- В панели управления хостингом создайте новую базу данных (например, city_portal).
- Создайте пользователя БД и дайте ему полные права на эту базу.
- Через инструмент phpMyAdmin выполните импорт файла `database.sql` (прилагается к проекту).

Шаг 3. Конфигурация подключения:
- Откройте файл `db.php` и измените параметры подключения на те, что выдал хостинг:
  $host = 'localhost'; (или IP сервера БД)
  $db   = 'имя_вашей_базы';
  $user = 'имя_пользователя_базы';
  $pass = 'пароль_от_базы';

Шаг 4. Права доступа к папкам:
- Установите права доступа (CHMOD 775 или 777) для папок:
  /img/          -- для загрузки фотографий к новостям.
  /sent_emails/  -- для сохранения текстовых копий обращений (feedback).

4. СТРУКТУРА ПРОЕКТА
--------------------
- index.php       -- Главная страница (Hero, 4 новости, AJAX-слайдер).
- all-news.php    -- Полный архив новостей с AJAX-фильтрацией.
- news.php        -- Страница конкретной новости и комментариев.
- search.php      -- Результаты поиска по сайту.
- admin_panel.php -- Единый центр управления (для Admin и Editor).
- auth_handler.php-- Обработчик всех форм (Вход, Регистрация, Feedback).
- fetch_data.php  -- API-модуль для AJAX подгрузки данных.
- db.php          -- Подключение к СУБД MySQL через PDO.
- style.css       -- Каскадные таблицы стилей (включая темную тему).
- script.js       -- Клиентские скрипты (Карта, Числа, Валидация).

5. ФОРМИРОВАНИЕ ОТЧЕТОВ
-----------------------
В панели администратора (вкладка Обращения) доступна кнопка генерации 
текстового отчета. При нажатии система формирует файл .txt со всеми 
входящими предложениями от жителей города.
